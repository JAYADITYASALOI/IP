# ENewsletter

**Project**: ENewsletter  
**Stack**: Java, Servlets, JSP, JSTL, MySQL, Maven, Tomcat

---

## Overview

ENewsletter is a simple newsletter web application that supports user registration, login, profile management, and article listing. The project uses JSP for views, servlets for controllers, and MySQL for persistence.

---

## Prerequisites

- **Java JDK** 8 or 11 installed and `JAVA_HOME` set.  
- **Maven** installed.  
- **MySQL** 5.7+ or 8.0+ running locally or accessible from the app.  
- **Tomcat 9** (recommended) for deployment (uses `javax.servlet`).  

---

## Project layout

**Important paths**

- **Source code**: `src/main/java/newsletter/`  
- **JSP views and webapp**: `src/main/webapp/`  
- **Resources**: `src/main/resources/` (`db.properties`, `app.properties`)  
- **SQL schema**: `sql/create_schema_custom.sql`  
- **Build file**: `pom.xml`  
- **README**: project root `README.md`
